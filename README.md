# AI 资讯追踪仓库

> 每天 8:30 自动汇总"昨日"全球 AI 资讯，按话题板块（LLM 发展 / 编程 Agent / 数字人）整理 Top 10，每周做汇总。

## 目录结构

```
AI资讯追踪/
├── 模板/                  # 模板文件（每日资讯模板、每周汇总模板）
├── 每日资讯/              # 按日期归档的每日资讯（YYYY-MM-DD.md）
├── 每周汇总/              # 按周归档的每周汇总（YYYY-WXX.md）
├── README.md              # 本文件
└── .gitignore
```

## 自动化约定

- **每日资讯**：每天 8:30 抓取昨日 00:00-23:59 的内容
- **每周汇总**：每周日早 8:30 汇总本周内容
- **每次更新后自动 commit + push** 到 remote
